# PHP E-Commerce Website (Academic Project) - HomeOfTheBoards

###HomeOftheBoards is a webiste like Facebook Marketplace/Gumtree where users can buy and sell used boards game from other people

* E-Commerce website created with PHP using SQL database.
* User Registration, products listing, Full CRUD operations
* PHP for server-side logic and MySQL as a Database
* Tech Stack: PHP, MySQL, HTML, CSS, JavaScript.

Was hosted using free website, but has since expired.
